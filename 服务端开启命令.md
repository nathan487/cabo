# Cabo 服务端启动命令

以下命令在 WSL Ubuntu 终端中执行。

## 重新构建并启动

```bash
cd "/mnt/c/Users/Admin/Desktop/Cabo GameObject/MuduoBaseGameServer"

cmake -S . -B build
cmake --build build --clean-first --target GameServer -- -j"$(nproc)"

cd build
LD_PRELOAD="$HOME/anaconda3/lib/libprotobuf.so.31" ./GameServer 8888
```

构建过程会自动根据仓库根目录的 `Proto/*.proto` 重新生成服务端 Protobuf C++ 代码。

## 只启动已构建的服务端

```bash
cd "/mnt/c/Users/Admin/Desktop/Cabo GameObject/MuduoBaseGameServer/build"
LD_PRELOAD="$HOME/anaconda3/lib/libprotobuf.so.31" ./GameServer 8888
```

如果提示端口 `8888` 已被占用，先停止旧服务端：

```bash
pkill -x GameServer
```

## 可选：Cloudflare 临时隧道

服务端启动后，在另一个 WSL 终端执行：

```bash
cloudflared tunnel --url http://127.0.0.1:8888
```

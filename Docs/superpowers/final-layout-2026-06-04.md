# GameScene UI 最终布局方案

## 布局总览

Canvas 分辨率：800x600，锚点在中心 (0,0)
可见范围：Y轴从 -300 到 +300

## 元素位置（从上到下）

| 元素 | Y坐标 | 高度 | 范围 | 说明 |
|------|-------|------|------|------|
| **HUD** | 280 | 40 | [260, 300] | 顶部信息栏，显示回合/轮数 |
| **对手区域** | 195 | 110 | [140, 250] | 对手的卡牌和信息 |
| **牌堆区域** | 50 | 160 | [-30, 130] | 抽牌堆和弃牌堆 |
| **抽牌预览** | 50 | 45 | [28, 73] | **与牌堆重叠**，仅在抽牌时显示 |
| **自己区域** | -177 | 110 | [-232, -122] | 自己的卡牌和信息 |
| **按钮面板** | -261 | 38 | [-280, -242] | 操作按钮，离底部20px |

## 间距验证

- HUD ↔ 对手区域：10px ✅
- 对手区域 ↔ 牌堆：10px ✅
- 牌堆 ↔ 自己区域：92px ✅
- 自己区域 ↔ 按钮面板：10px ✅
- 按钮到底部边界：20px ✅

## 关键调整

### 1. 缩小元素高度
- HUD: 50 → 40
- 对手区域: 120 → 110
- 牌堆: 180 → 160
- 自己区域: 120 → 110
- 按钮: 42 → 38

### 2. 抽牌预览特殊处理
- 放置在牌堆中心（y=50），与牌堆重叠显示
- 只在抽牌后显示，平时隐藏
- 避免占用额外的垂直空间

### 3. 按钮面板安全位置
- y=-261，确保离底部边界至少20px
- 所有按钮都在可见范围内

## 代码关键点

### CreatePlayerArea 参数
```csharp
CreatePlayerArea(parent, name, pos, isSelf, height=110)
```
- 高度参数可调整
- 文字位置根据高度自动调整：`textOffsetY = height/2 - 15`

### 按钮尺寸
```csharp
MakeButton(parent, label, pos, new Vector2(width, 34), color)
```
- 按钮高度统一为 34px

### 牌堆元素
```csharp
DrawPile: 100x110
DiscardPile: 100x110
```
- 缩小以适应整体布局

## 测试要点

1. ✅ 对手信息完全可见（名称、分数、4张卡）
2. ✅ 牌堆显示清晰
3. ✅ 自己的卡牌完全可见，不被截断
4. ✅ 操作按钮在底部清晰可见
5. ✅ 抽牌时预览显示在牌堆上方，不遮挡其他元素

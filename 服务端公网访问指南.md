# 服务端公网访问指南

## 1. 启动服务端

```bash
cd MuduoBaseGameServer/build
LD_PRELOAD=$HOME/anaconda3/lib/libprotobuf.so.31 ./GameServer 8888
```

服务端监听 `127.0.0.1:8888`

## 2. 内网穿透 (ngrok) — 让外网客户端连接

### 一次性设置（仅首次）

1. 打开 https://dashboard.ngrok.com/signup 注册免费账号
2. 登录后进入 https://dashboard.ngrok.com/get-started/your-authtoken
3. 复制你的 authtoken
4. 在 WSL 终端执行：
```bash
~/.local/bin/ngrok config add-authtoken 你的authtoken
```

### 每次启动

```bash
~/.local/bin/ngrok tcp 8888
```

启动后会显示：
```
Forwarding  tcp://0.tcp.ngrok.io:12345 -> localhost:8888
```

### 客户端连接

外网玩家使用 ngrok 显示的地址连接，例如：
```
0.tcp.ngrok.io:12345
```

Unity 客户端修改 GameBootstrap 上的 `serverHost` / `serverPort`。

CLI 客户端直接输入 `0.tcp.ngrok.io:12345`。

## 3. 局域网访问（同一网络）

如果玩家在同一 WiFi/局域网，直接使用 WSL 的 IP：

```bash
# 查看 WSL IP
hostname -I
# 例如: 172.24.165.161
```

客户端连接 `172.24.165.161:8888`。

注意：需要在 Windows 防火墙开放 8888 端口：
```powershell
# 以管理员身份运行 PowerShell
New-NetFirewallRule -DisplayName "Cabo Game Server" -Direction Inbound -Port 8888 -Protocol TCP -Action Allow
```

## 4. 验证服务可用

```bash
# 检查服务端是否在运行
ss -tlnp | grep 8888

# 检查 ngrok 隧道
curl -s http://127.0.0.1:4040/api/tunnels
```
